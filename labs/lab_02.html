Lab 2 Create a Project

Goals

  • Learn how to create a Git repository from scratch.

Create a “Hello, World” program

Starting in the empty working directory, create an empty directory named “hello”, then create a file named hello.r with the contents below.

Execute:

mkdir hello
cd hello

hello.r

puts "Hello, World"

Create the Repository

You now have a directory with a single file. To create a Git repository from that directory, run the git init command.

Execute:

git init

Output:

$ git init
Initialized empty Git repository in /path/to/your/hello/.git/

Add the program to the repository

Now let’s add the “Hello, World” program to the repository.

Execute:

git add hello.r
git commit -m "First Commit"

Output:

$ git add hello.r
$ git commit -m "First Commit"
[main (root-commit) f7c41d3] First Commit
 1 file changed, 1 insertion(+)
 create mode 100644 hello.r
